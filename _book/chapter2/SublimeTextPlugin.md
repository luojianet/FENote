* rem-unit
```
参数配置文件：
Sublime Text -> Preferences -> Package Settings -> rem-unit
fontsize - html元素font-size值，默认为16。
precision - px转rem的小数部分的最大长度，默认为8。
leadingzero - 是否保留前导0，默认不保留。
exts - 启用此插件的文件类型。默认为：[".css", ".scss", ".less", ".sass", ".styl"]。
```
